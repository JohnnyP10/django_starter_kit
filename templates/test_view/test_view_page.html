Get started with Test Driven Development

This page is here to help you get you up and running with unit testing. You need to send a post request to this page, that will show up in the Output box below. Go to the django_starter_kit/test.py file and look at the test_page_output_message function. What is it looking for, and how do you get it to show up in the template?

start by running the following command in the command line: 

python -Wi manage.py test django_starter_kit/

For more information look at the test_view_page.html template and the views.py file in the django_starter_kit app.
